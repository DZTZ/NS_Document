🛠️ NS_Replay 插件 使用指南

欢迎使用本插件！以下是蓝图节点讲解，帮助您快速集成 UE 回放功能。

开始录制回放 (Start Recording)

该节点用于启动当前游戏会话的录制过程。您可以为每个录制文件指定唯一的名称，以便后续进行查找和播放。

蓝图节点图：

输入参数 (Input) - 左侧引脚

参数名称 数据类型 用途描述
执行引脚 (顶端) Exec **触发** 录制逻辑，调用后系统将立即开始记录游戏数据。
Replay Name String **必填：** 指定本次录制的文件名称（ID）。建议使用唯一标识符，避免覆盖旧的回放文件。

输出参数 (Output) - 右侧引脚

参数名称 数据类型 用途描述
执行引脚 (顶端) Exec 录制指令成功发出后立即触发。注意：这仅表示“开始录制”的操作已执行。
Return Value Boolean **返回值：** 标识录制任务是否成功启动。返回 `True` 表示录制已开始，返回 `False` 可能是由于权限、磁盘空间或已有正在进行的录制任务。

停止录制回放 (Stop Recording)

该节点用于停止当前正在进行的游戏录制。调用此节点后，系统将结束录制并保存回放文件。

蓝图节点图：

输入参数 (Input) - 左侧引脚

参数名称 数据类型 用途描述
执行引脚 (顶端) Exec **触发** 停止录制逻辑。

输出参数 (Output) - 右侧引脚

参数名称 数据类型 用途描述
执行引脚 (顶端) Exec 停止录制指令发出后触发后续逻辑。
Return Value Boolean **返回值：** 标识是否成功停止录制。如果当前并没有正在进行的录制任务，可能会返回 `False`。

获取回放列表 (Get Replay List)

该节点用于从存储设备中读取所有可用的回放文件信息列表。

蓝图节点图：

输出参数 (Output)

参数名称 数据类型 用途描述
Out Replay List Array (Struct) **核心数据：** 包含所有回放信息的数组。每个元素包含回放名称、路径等结构体数据。
Return Value Boolean **返回值：** 是否成功获取到列表（即使列表为空，只要查询操作成功也会返回 True）。

数据结构：Replay Info (通过 Break 节点查看)

  • Replay Name (String): 录制时指定的唯一名称。
  • Replay Path (String): 文件在磁盘上的物理存储路径。

播放回放 (Play Replay)

该节点用于启动回放功能。系统将寻找与输入名称匹配的录制文件并切换到回放观察模式。

蓝图节点图：

输入参数 (Input)

参数名称 数据类型 用途描述
执行引脚 Exec 触发加载并播放回放的逻辑。
Replay Name String **必填：** 需要播放的回放文件名称（例如：myReplay_2025-12-06 19-56）。

输出参数 (Output)

参数名称 数据类型 用途描述
执行引脚 Exec 指令发出后立即触发。
Return Value Boolean **返回值：** 如果成功找到文件并开始播放，返回 True；若文件不存在或加载失败则返回 False。

删除回放 (Delete Replay)

该节点用于清理不再需要的录制数据。注意：删除操作是不可逆的。

蓝图节点图：

输入参数 (Input)

参数名称 数据类型 用途描述
执行引脚 Exec 触发删除文件的逻辑。
Replay Name String **必填：** 想要删除的目标回放文件名称。

输出参数 (Output)

参数名称 数据类型 用途描述
执行引脚 Exec 删除操作尝试完成后触发。
Return Value Boolean **返回值：** 文件成功删除返回 True；若文件未找到或由于权限等原因删除失败则返回 False。

获取回放总时长 (Get Replay Total Duration)

在回放过程中调用，返回当前正在播放的回放视频的总持续时间。常用于初始化播放进度条的最大值。

蓝图节点图：

输入参数 (Input)

参数名称 数据类型 用途描述
执行引脚 Exec 触发查询总时长的指令。

输出参数 (Output)

参数名称 数据类型 用途描述
执行引脚 Exec 查询指令执行完毕后触发。
Return Value Float **返回值：** 回放的总时长（以秒为单位）。若未在回放中，通常返回 0.0。

获取当前回放时间 (Get Current Replay Time)

用于实时追踪回放进度。可以配合 Tick 或计时器使用，以同步更新 UI 上的进度条和时间文字显示。

蓝图节点图：

输入参数 (Input)

参数名称 数据类型 用途描述
执行引脚 Exec 触发获取当前播放进度的指令。

输出参数 (Output)

参数名称 数据类型 用途描述
执行引脚 Exec 指令执行完毕后触发。
Return Value Float **返回值：** 当前回放所在的秒数位置（例如：15.5 秒）。

跳转至时间点 (Seek To Time)

允许玩家通过进度条或输入数值，直接跳转到回放中的特定时间位置。常用于快速定位精彩瞬间。

蓝图节点图：

输入参数 (Input)

参数名称 数据类型 用途描述
执行引脚 Exec 触发跳转指令。
Time in Seconds Float **目标时间：** 想要跳转到的时间点（单位：秒）。请确保数值在 0 到总时长之间。

输出参数 (Output)

参数名称 数据类型 用途描述
执行引脚 Exec 跳转请求发出后触发。
Return Value Boolean **返回值：** 跳转操作是否成功启动。

设置回放播放速度 (Set Replay Playback Speed)

动态调整回放的播放速率。例如，设置为 2.0 表示两倍速快进，设置为 0.5 表示慢动作播放。

蓝图节点图：

输入参数 (Input)

参数名称 数据类型 用途描述
执行引脚 Exec 触发速度变更指令。
Playback Speed Float **播放速率：** 默认值为 1.0（正常速度）。
  • > 1.0: 加速播放
  • < 1.0: 慢动作
  • 0.0: 暂停（取决于具体实现）

输出参数 (Output)

参数名称 数据类型 用途描述
执行引脚 Exec 速率设置成功后触发。
Return Value Boolean **返回值：** 速度修改是否成功应用。

在回放过程中调用，将当前回放画面和进度冻结。常用于实现播放器的暂停功能。

蓝图节点图：

输入参数 (Input)

输出参数 (Output)

Return Value Boolean **返回值：** 是否成功执行暂停。若当前未处于回放状态则返回 False。

输出参数 (Output)

Return Value Boolean **返回值：** 是否成功恢复播放。

蓝图节点图：

输出参数 (Output)

参数名称 数据类型 用途描述
Return Value Boolean **返回值：** 如果当前正处于回放模式返回 True；若在正常游戏逻辑中则返回 False。
